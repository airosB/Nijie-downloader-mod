Nijie downloader

Default (single)
nijie/${userName}(${userId})/${title}(${id})
Default (multi)
nijie/${userName}(${userId})/${title}(${id})/${page}
Variables Description
${id} Illust ID
${title} Illust Title
${userId} User ID
${userName} User Name
${page} Page Number
  • '\/:*?"<>|~' is replaced '_'.
  • ${page} must be located last path element.
